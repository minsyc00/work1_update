# 0630重要指标及优化

不重新运行，只看上次 50x50 报告。我的评价是：**结果达标，但还不是很“优雅”**。它已经解决了最关键的“左侧大块区域不执行”问题，主规划执行完整、无跳过、无碰撞越界；但从覆盖质量、重复覆盖、路径利用率和转向代价看，还有明显优化空间。

**结果亮点**

- 总覆盖率 `coverage_fraction = 0.9908`，达到目标 `0.99`。
- 主区域执行状态 `region_execution_status = complete`。
- `skipped_region_count = 0`，23 个可行主区域全部执行。
- 越界、障碍碰撞、运动学不可行都是 `0`。
- 左侧大块垂直往复扫描已经进入最终执行路径。

**主要问题**

1. **覆盖率达标有一部分依赖非正式扫描路径**
   - `coverage_fraction = 0.9908`
   - `cover_only_coverage_fraction = 0.9788`
   - 差值约 `1.2%`，说明最终达到 0.99 有一部分来自 transit/turn 的扫掠覆盖，而不是纯覆盖线。
   - 这在工程上可接受，但论文式覆盖评价里最好把“正式覆盖”和“转场附带覆盖”分开要求。

2. **路径利用率偏低**
   - 总路径长度 `1857.60 m`
   - 覆盖段长度 `1029.03 m`
   - 转场段长度 `828.57 m`
   - 覆盖段占比只有 `55.4%`，转场占比 `44.6%`，说明路径中接近一半不是正式覆盖作业。

3. **重复覆盖偏高**
   - 主路径重复重叠长度 `311.44 m`
   - 重复占总路径约 `16.8%`
   - 重复相对转场长度比例 `37.6%`
   - 跨艇重叠 `126.94 m`，其中 transit `65.26 m`，turn `51.68 m`，cover 只有 `10 m`。也就是说，重复主要来自连接路径和转弯，而不是覆盖线本身。

4. **转向代价较大**
   - 总转向角 `511.91 rad`
   - 转弯次数 `59`
   - 平均每次转向约 `8.68 rad`，说明很多连接/回转不是简单小角度调整，而是包含较大绕行或多段转向。

5. **艇间负载不均**
   - 各艇总路径长度：
     - USV0: `407.57 m`
     - USV1: `714.85 m`
     - USV2: `735.18 m`
   - 负载不均衡指标 `0.529`，USV0 明显偏闲，USV1/USV2 承担更多路径和连接。

6. **回缩和残余补扫仍偏多**
   - 自适应回缩 pass 数 `196`
   - 总回缩长度 `289.90 m`
   - 扩展回缩 `30` 次，失败回缩 `18` 次
   - residual backfill 追加了 `2` 个残余补扫，仍剩 `3` 个 residual component，残余面积比例 `0.0092`。

**值得优化的方向**

- 提高 `cover_only_coverage_fraction`，让目标覆盖更多来自正式往复扫描线，而不是转场附带扫掠。
- 降低转场长度占比，减少区域间来回穿插。
- 降低重复覆盖，尤其是 transit/turn 引起的重复。
- 降低总转向角和大角度绕行。
- 改善多艇负载均衡，让 USV0 分担更多区域或残余补扫。
- 减少回缩次数和回缩长度，说明区域形状、扫描方向和 U-turn 布置还可以更协调。

**优化代价很小的项**

- 报告层增加 `cover_only_target_status`，不要只看 `coverage_fraction` 是否达标。
- pattern 评分里提高“回缩损失”“转向角”“重复覆盖”的惩罚权重。
- connector 评分里加重已有路径重叠惩罚，尤其是 transit/turn overlap。
- 负载分配后做一次轻量 region swap，把 USV1/USV2 的边界区域尝试转给 USV0。
- residual 补扫前按“新增覆盖面积 / 新增路径长度”筛选，避免为了很小残余插入代价很高的补扫。
- 对残余补扫候选优先选本艇附近、少转弯、少重复的短线，而不是只看能否连接成功。
- 把扫描方向选择从“覆盖长度优先”改成“覆盖长度 + 回缩损失 + U-turn 数量 + 连接端点质量”综合评分。

**代价较大的优化**

- 把区域分配、扫描方向、入口出口和连接路径做联合优化。
- 全局多艇协同排序，显式最小化交叉和重复路径。
- 重新设计 residual planner，使它不是事后补丁，而是主区域规划的一部分。
- 做更强的运动学平滑或轨迹优化，降低总转向角和绕行长度。

一句话总结：**这次结果已经“可用且达标”，但还偏“能覆盖上”，不是“覆盖得经济”。下一步最划算的优化，是优先减少转场重复、改善艇间负载，并把 cover-only 覆盖率作为更严格的主指标。**
