# static_obstacle_map_400x400_mixed_obstacles

## 用途

该地图是 `400m x 400m` 多形态静态障碍全覆盖路径规划实验环境资产。文件只描述任务区域和静态障碍物，不包含无人艇初始状态、覆盖路径、轨迹参考或仿真结果。

本版本按当前大块凸区域分解、倾斜往复扫描、区域间连接和 residual 补扫算法做适配。地图仍保留 400m 大尺度、圆形、椭圆、凸多边形、浅凹多边形和 S 型障碍特征，但障碍数量控制在 8 个，避免使用长距离凹形多边形屏障、密集小礁石和北侧碎片簇，减少区域碎片化和 connector 搜索长尾。

## 任务区域

- 区域类型：矩形水域。
- 坐标范围：`[0, 400] x [0, 400]`。
- 单位：`m`。
- 原点：左下角 `(0, 0)`。

## 覆盖与运动约束

- 覆盖足迹：矩形。
- 覆盖足迹长度：`length_lf = 4.0 m`。
- 覆盖足迹宽度：`width_wf = 2.0 m`。
- 无人艇模型：`3-DOF`。
- 最小转弯半径：`Rmin = 2.0 m`。
- 推荐无人艇数量：`5`。
- 推荐重叠率：`rho = 0.2`。
- 推荐安全距离：`d_safe = 1.0 m`。

## S 型障碍说明

原先的 `s_curve_barrier_c_01` 是一个 14 顶点长凹多边形，容易把自由空间切成复杂细碎区域。上一版用三段不重叠椭圆弧链近似 S 型障碍；当前版本进一步改为单个 `s_curve_shallow_polygon_c_01` 浅凹 S 型多边形。

该多边形顶点数较少、凹入较浅，视觉上形成较短、平缓的 S 型障碍带；同时不横向切断 400m 地图的中央水域，更适合当前区域分解和倾斜往复扫描生成。

## 适配当前算法的设计说明

- 删除 L/U 形凹多边形、密集小礁石簇和北侧碎片障碍，避免障碍坐标切分产生大量碎片区域。
- 保留一个浅凹多边形 `shallow_concave_island_nw_01`，凹口较宽且凹入较浅，不形成 U 形死胡同。
- 所有主要障碍保持非重叠，并与任务边界保持明显距离。
- 中央、东西两侧和北侧都保留大块自由水域，便于生成大矩形、梯形或凸多边形区域。
- S 型障碍使用单个低顶点数浅凹多边形，保留弯曲边界测试价值，但不横向切断整张地图。
- 障碍类型全部来自当前地图加载器支持的类型：`rectangle`、`ellipse`、`circle`、`polygon`。

## 不包含内容

- 不包含无人艇初始状态。
- 不包含动态障碍物。
- 不包含覆盖路径。
- 不包含轨迹参考。
- 不包含仿真结果。
