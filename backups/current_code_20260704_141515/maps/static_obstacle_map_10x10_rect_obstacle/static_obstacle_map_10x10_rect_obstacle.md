# static_obstacle_map_10x10_rect_obstacle

## 用途

该地图用于验证静态障碍物场景下的多无人艇覆盖路径规划基础流程。地图只包含任务区域和一个矩形静态障碍物，不包含无人艇初始状态、路径结果、覆盖条带或仿真轨迹。

## 任务区域

- 区域类型：矩形水域。
- 坐标范围：`[0, 10] x [0, 10]`。
- 单位：`m`。
- 原点：左下角 `(0, 0)`。

## 覆盖与运动约束

- 覆盖足迹：矩形。
- 覆盖足迹长度：`length_lf = 4.0 m`。
- 覆盖足迹宽度：`width_wf = 2.0 m`。
- 无人艇模型：`3-DOF`。
- 最小转弯半径：`Rmin = 2.0 m`。
- 推荐重叠率：`rho = 0.2`。
- 推荐安全距离：`d_safe = 0.5 m`。

## 静态障碍物

| ID | 类型 | 参数 |
| --- | --- | --- |
| `rect_obstacle_01` | rectangle | center=`(5.0, 5.0)`, size=`2.0 x 2.0`, yaw=`0 deg` |

## 不包含内容

- 不包含无人艇数量或初始位姿。
- 不包含动态障碍物。
- 不包含规划路径。
- 不包含轨迹参考或闭环仿真结果。
